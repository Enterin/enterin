Includiamo EnterIN.JS nella nostra pagina web

Per includere EnterIN.JS nella nostra pagina web abbiamo bisogno di 2 righe di codice:

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
<script rel="js/enterin" src="enterin/enterin.js"></script>



* EnterIN.JS va incluso nella root del sito e va richiamato con "enterin/enterin.js"

Includere jQuery e EnterIN.JS all'interno del tag <head>:

<!DOCTYPE html>
	<html>
	<head>
		<meta charset="utf-8">
		<title>Page Title</title>

		<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
		<script rel="js/enterin" src="enterin/enterin.js"></script>

	</head>
	...

Facciamolo funzionare!

Una volta averlo incluso nella nostra pagina web, EnterIN.JS è pronto essere richiamato!

Vediamo insieme le classi e gli attributi per far funzionare EnterIN.JS correttamente.

<section data-enterin="true" data-enterin-type="page">
	<figure class="enterin-slide" data-enterin-to="1"><img src="shape1.png" alt=""></figure>
	<figure class="enterin-slide" data-enterin-to="2"><img src="shape4.png" alt=""></figure>
	<figure class="enterin-slide" data-enterin-to="3"><img src="shape3.png" alt=""></figure>
	<figure class="enterin-slide" data-enterin-to="4"><img src="shape5.png" alt=""></figure>
	<figure class="enterin-slide" data-enterin-to="5"><img src="shape2.png" alt=""></figure>
</section>

Spieghiamo adesso il markup html5 di cui ha bisogno EnterIN.JS:

data-enterin="true"		(required)

Tale attributo comunica ad EnterIN.JS che l'elemento <div> o <section> contengono le istruzioni necessarie 
per far girare EnterIN.JS

class=".enterin-slide"		(required)

Tutti gli elementi/slide/pagine/immagini all'interno del tag <div> o <section> con attributo data-enterin="true"

data-enterin-to="#"	(optional)

Tale attributo determina la posizione # della slide e nel caso viene cliccata o tappata, mostra la slide #.

data-enterin-type="page"	(optional)

Tale attributo determina se EnterIN.JS debba interpretare il markup come nativo o come pagina.

Configurazione

Una volta creato il nostro markup, non ci resta che configurare la velocità dell'animazione e lo stile della nostra applicazione web

La prima cosa da fare è quindi aprire il file enterin/config.enterin.css:

/*
*
* 	Modifichiamo la velocità ed il tipo di effetto di EnterIN.JS
*
* 	@enterin-effect-speed: 		CSS3 transition-time value
*	@enterin-effect-ease: 		CSS3 transition-ease value
*
*/

@enterin-effect-speed: 		1.5s;

@enterin-effect-ease: 		ease-in-out;

/*
*
* 	Ora possiamo sovrascrivere una o più regole css senza toccare i file core.enterin.css e style.enterin.css
*
*	.enterin-body-css: 			css style per il tag body
*	.enterin-wrapper-css: 			css style per element[data-enterin="true"]
*	.enterin-slide-css: 			css style per element[data-enterin="true"] .enterin-slide
*
*/

.enterin-body-css {
	
	/* css code here */

}

.enterin-wrapper-css {

	/* css code here */

}


.enterin-slide-css {

	/* css code here */
	
}

Customizzazione

Terminata la parte di configurazione passiamo a quella di customizzazione!

Per customizzare lo stile della nostra applicazione web abbiamo solamente bisogno di modificare il file custom.enterin.css

Eventi

Vediamo adesso come gestire gli eventi di EnterIN.JS

Mouse Enter:

Tale evento si attiva quando passiamo con il mouse su una slide assegnando a tale slide la classe mouseenter

Per intervenire sull'evento ed assegnarne lo stile possiamo quindi assegnare la seguente regola nel custom.enterin.css:

element[data-enterin="true"] .enterin-slide.mouseenter {
	/* le nostre regole css */
}

EnterIN.JS Active:

Tale evento si attiva sulla slide corrente che stiamo visualizzando assegnando la classe enterin-active

Per intervenire sull'evento ed assegnarne lo stile possiamo quindi assegnare la seguente regola nel custom.enterin.css:

element[data-enterin="true"] .enterin-slide.enterin-active {
	/* le nostre regole css */
}

Mouse, tastiera e... dita!

EnterIN.JS sfrutta anche eventi inizializzati dai click del mouse, lo scorrimento con la rotella del mouse, la pressione di un tasto della tastiera e la pressione del touch con le dita

Vediamoli insieme:

MOUSE:

	rotella giu:						+1 indice della slide corrente. Se siamo all'ultima slide ritorna alla 1.
	rotella su: 						-1 indice della slide corrente. Se siamo alla prima slide va all'ultima.
	tasto centrale del mouse: 				Attiva la modalità "griglia" per visualizzare tutte le slide in 2 colonne.
	 							Premendolo nuovamente la disattiva


TASTIERA:

	freccia destra/giu:					+1 indice della slide corrente. Se siamo all'ultima slide ritorna alla 1.
	freccia sinistra/su: 					-1 indice della slide corrente. Se siamo alla prima slide va all'ultima.
	tasto G: 						Attiva la modalità "griglia" per visualizzare tutte le slide in 2 colonne.
	 							Premendolo nuovamente la disattiva

TOUCH:

	swipe sinistra:						+1 indice della slide corrente. Se siamo all'ultima slide ritorna alla 1.
	swipe destra: 						-1 indice della slide corrente. Se siamo alla prima slide va all'ultima.
	tap 2 dita: 						Attiva la modalità "griglia" per visualizzare tutte le slide in 2 colonne.
	 							Ripetendo il gesto la disattiva

Avanzate
Controlliamo EnterIN.JS

Vediamo ora come possiamo richiamare tali eventi assegnandoli ad elemento o una funzione esterna.

Possiamo decidere di assegnare ad un elemento l'attributo data-enterin-to="#" per far si che al click ci si possa spostare all'indice desiderato.

<a href="#" data-enterin-to="10">CLICCAMI PER ANDARE ALLA SLIDE 10</a>

L'equivalente dell'esempio sopra può essere sfruttato anche nel seguente modo mediante l'ausilio di Javascript:

<script>  $.EnterIN.changeSlide(10); </script>
